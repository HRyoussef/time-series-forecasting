# Time Series Forecasting Project

## Overview

This project explores, analyzes, and forecasts time series data using classical statistical forecasting techniques and modern forecasting libraries.

The project is divided into three main parts:

1. Exploratory Time Series Analysis
2. Forecasting with ARIMA Models
3. Forecasting with Prophet

## Objectives

* Understand time series analysis challenges
* Perform exploratory data analysis on temporal datasets
* Build ARIMA and SARIMA forecasting models
* Evaluate forecasting performance
* Compare classical forecasting approaches with Prophet
